PRIMEIROS COMANDOS COM GIT

- listar as pastas
ls

- entrar na pasta especificada
cd nome da pasta

- criar repositório
mkdir nome-repositorio

- iniciar o repositorio
git init

- mostrar arquivos ocultos
ls -a

- voltar um nível
cd ..

- 
git config --global user.email "email@email.com"

git config --global user.name digitarNickName

- commit do arquivo
git add *

git commit -m "texto/mensagem"

- ajuda a monitorar o status dos arquivos
git status
